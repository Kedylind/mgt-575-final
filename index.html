Problem and Opportunity

Decision fatigue in urban discovery

Core user pain

  • Maps and review apps favor popularity over personal context and vibe fit.
  • Social apps are great for inspiration but weak for structured decision-making.
  • Friend recommendations are trusted but not searchable or persistent.

Tailr hypothesis

  • Best recommendations sit at the intersection of taste, mood, and local trust.
  • Natural-language intent should replace brittle search-and-filter workflows.
  • Social check-ins and curation should feed a persistent discovery graph.
Solution

What Tailr Delivers

1

Intent Capture

User asks in natural language, such as "cozy bars in Williamsburg".

UI position: top search + Ask TailrBot
2

Intent Parsing

Client-side NLU extracts neighborhood, type, vibe, and price intent.

AI role: rule-based semantic extraction
3

Retrieval and Ranking

Server scoring blends type match, proximity, tag overlap, price, and rating.

UI position: map pins + ranked suggestion cards
4

Action Layer

Results include human-readable reasons, then route to save, navigate, or check-in.

Results and Analysis

Testing outcomes: wins and failure modes

Observed strengths

  • Top use cases (type + neighborhood + vibe) perform strongly in TailrBot.
  • Map + chat integration creates a differentiated spatial discovery experience.
  • Swipe-to-save lets users build 8-15 place lists in under 5 minutes.

Current limitations

  • Ambiguous/negation-heavy queries still break rule-based parsing.
  • Real-time data coverage (live hours/availability) is still incomplete.
  • Thin follow graph at launch causes feed cold-start issues.

Market opportunity

  • Urban users still rely on fragmented tools and high-friction research loops.
  • Incumbents optimize for popularity and utility, not nuanced context fit.
  • Tailr focuses on the discovery-and-intent layer incumbents underserve.
  • Secondary value: organic visibility path for local businesses.

Competitive landscape

  • Yelp: massive review base; weak social trust graph and AI chat UX.
  • Google Maps: real-time utility; weaker vibe/occasion understanding.
  • Foursquare/Infatuation/Fever: partial strengths, but fragmented use cases.
  • Tailr position: social trust + conversational intent + gamified discovery.
